# 🌍 Weather + AQI Map Dashboard

An interactive web app that shows **real-time weather 🌡️ and Air Quality Index 🌫️** for any location in the world.

Built using **HTML, CSS, JavaScript, and Leaflet Maps**.

---

## 🚀 Features

* 🌍 Interactive world map (click anywhere)
* 🔍 Search any city (Enter key supported)
* 🌡️ Live weather data (temperature + condition)
* 🌫️ Real AQI values (0–500 scale)
* 🌙 Dark / Light mode toggle
* 📱 Fully responsive (works on mobile + desktop)
* 📍 Marker appears on selected location

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Leaflet.js (for maps)
* OpenWeather API (weather data)
* WAQI API (Air Quality Index)

---

## 🔑 API Setup

### 1. Weather API

Get your key from: https://openweathermap.org/api

Replace in code:

```js
const apiKey = "YOUR_OPENWEATHER_API_KEY";
```

---

### 2. AQI API

Get your token from: https://aqicn.org/data-platform/token/

Replace in code:

```js
const token = "YOUR_WAQI_TOKEN";
```
